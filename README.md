# PixelStub Frontend

React frontend for building placeholder image URLs against the PixelStub Worker API.

## Scripts

- `npm run dev` starts Vite.
- `npm run build` creates a production build.
- `npm test` runs component tests.

## Environment

Create `.env.local` when the API is not on the default local Worker URL:

```bash
VITE_API_BASE_URL=http://localhost:8787
```
